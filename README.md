## Description

A simple Graph API on the Node.js platform using Typescript. CouchDB is used for data storage.

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ docker-compose up

$ npm run start
```

## Usage

[http://localhost:5000/graphql](http://localhost:5000/graphql)
